# Mukesh | Staff AI/ML Engineer @ Genesys

Welcome to my GitHub portfolio. 

I am a Data Science graduate from Trinity College Dublin, currently based in Dublin, Ireland, and originating from Chennai, India.

I specialize in Artificial Intelligence, Large Language Models (LLMs), and autonomous AI systems, currently working as a Staff AI/ML Engineer at Genesys.

### Professional Overview

- 🔭 **Current Role:** Architecting and developing next-generation AI Agents and LLM-powered applications at Genesys.
- 🌱 **Focus Areas:** Large Language Models, Generative AI, Retrieval-Augmented Generation (RAG), AI Agent orchestration, and scalable machine learning infrastructure.
- 👯 **Collaboration:** Open to collaborating on open-source AI frameworks, innovative LLM research, and advanced machine learning initiatives.
- 🥅 **2026 Objectives:** Deepen contributions to open-source Generative AI projects and push the boundaries of autonomous agent capabilities.
- ⚡ **Interests:** Outside of engineering, I enjoy reading, playing basketball, and staying up-to-date with the latest advancements in AI.
